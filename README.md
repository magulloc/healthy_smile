
Healthy Smile

Genera partículas animadas en pantalla al detectar que la cara capturada por la cámara abre la boca. 
Una experiencia interactiva desarrollada con p5.js, ml5.js y p5.sound.

---

Cómo ejecutar

1. Descarga o clona este repositorio:

        git clone https://github.com/magulloc/healthy_smile.git
   
2. Instala las dependencias con npm
   
        npm install
   
3. Ejecutar la aplicación.
        
        Abrir la carpeta de la aplicación de un terminal y ejecutar: npm start
        
---

Requisitos

- Navegador moderno (Chrome, Firefox o Edge).
- Permisos de cámara activados.
- Conexión a internet (para cargar las librerías desde CDN).

---

Tecnologías utilizadas

- p5.js (https://p5js.org/)
- p5.sound (https://p5js.org/reference/#/libraries/p5.sound)
- ml5.js (https://ml5js.org/)
